# Sentinel

When you select install/reinstall Sentinel chain you need to config the port forwarding

![Set up port forwarding pop up](/images/port-forwarding/sentinel/set-up-port-forwarding-pop-up.png)

## Step 1: Get IP Address

### 1. Open Command Prompt on the computer

![Step 1.1](/images/port-forwarding/sentinel/step1.1.png)

### 2. Open Command Prompt on the computer
```js copy
  ipconfig /all
```

![Step 1.2](/images/port-forwarding/sentinel/step1.2.png)

### 3. Record the following: IPv4 Address, Subnet Mask, Default Gateway, and DNS Servers.

![Step 1.3](/images/port-forwarding/sentinel/step1.3.png)

## Step 2: Set Up The Static IP Address

### 1. Open the Run dialog box with the WIN+R keyboard shortcut, enter ncpa.cpl, and select OK to open Network Connections.

![Step 2.1](/images/port-forwarding/sentinel/step2.1.png)

### 2. Right-click or tap-and-hold the connection that has the same name as the one you identified in Command Prompt. For example, Ethernet0.

![Step 2.2](/images/port-forwarding/sentinel/step2.2.png)

### 3. Select Properties from the menu.

![Step 2.3](/images/port-forwarding/sentinel/step2.3.png)

### 4. Choose Internet Protocol Version 4 (TCP/IPv4) from the list, then select Properties.

![Step 2.4](/images/port-forwarding/sentinel/step2.4.png)

### 5. Select Use the following IP address.

![Step 2.5](/images/port-forwarding/sentinel/step2.5.png)

### 6. Enter the details you copied from Command Prompt: IP address, subnet mask, default gateway, and DNS servers. 

- For IP address: It must be a static IP. The last 2 digit should be the number in the range [150-254] to increase the ability that it is not assigned to other machine when your machine disconnect to the internet.

![Step 2.6](/images/port-forwarding/sentinel/step2.6.png)

### 7. Choose OK when you're done.

![Step 2.7](/images/port-forwarding/sentinel/step2.7.png)

## Step 3: Set Up Port Forwarding

### 1. Access the router as an administrator

![Step 3.1](/images/port-forwarding/sentinel/step3.1.png)

### 2. Locate the port forwarding options. These are different for every router but might be called something like Port Forwarding, Port Triggering, Applications & Gaming, or Port Range Forwarding. These might be buried within other categories of settings like Network, Wireless, or Advanced.

![Step 3.2](/images/port-forwarding/sentinel/step3.2.png)

### 3. Add New Port Redirection by button “Add”. Type 7777-7778 in port range and the same number under both the Public and Private Port

![Step 3.3](/images/port-forwarding/sentinel/step3.3.png)

### 4. Choose “TCP/UDP” from the list of Protocol dropdown

![Step 3.4](/images/port-forwarding/sentinel/step3.4.png)

### 5. Enter the IP address that you updated in the previous step

![Step 3.5](/images/port-forwarding/sentinel/step3.5.png)

### 6. Enable the port forwarding rule with an Enable or On option then Apply it

![Step 3.6](/images/port-forwarding/sentinel/step3.6.png)
