# Uninstallation

## Step 1

Go to the Settings page on your machine. Search for **Hivello** in the Apps > Installed apps

![Go to installed app Hivello](/images/uninstallation/go-to-installed-app-hivello.png)

## Step 2

Select the **more** option and sen select **Uninstall** application. Wait for the application collecting its file and uninstall itself.

![Uninstall application](/images/uninstallation/uninstall-application.png)
