# Uninstallation

## Step 1

To uninstall the Hivello application, first, search for it in the system search box. Then, select the **Uninstall** button. Alternatively, you can double-click on the downloaded Hivello file and then select the **Next** button to initiate the uninstallation process.

### Option 1: Uninstall from the application

![Uninstall from the application](/images/uninstallation/uninstall-from-the-application.png)


### Option 2: Uninstall from the downloaded file

![Uninstall from the downloaded file](/images/uninstallation/uninstall-from-the-downloaded-file.png)

## Step 2

Select the elements that you would like to uninstall, then select **Uninstall** button and wait for the uninstallation process is running

- Uninstall Windows Subsystem for Linux

![Uninstall Windows Subsystem for Linux](/images/uninstallation/uninstall-windows-subsystem-for-linux.png)

- Uninstall Hivello Deepin

![Uninstall Hivello Deepin](/images/uninstallation/uninstall-hivello-deepin.png)

## Step 3

After the Hivello app has been successfully uninstalled, your machine will need to be restarted. Please restart your machine to ensure the selected elements are completely uninstalled.

![Step 3](/images/uninstallation/step-3.png)
