# Installation

This section will guide you through the steps to install the Hivello application

## Prerequisites

- OS: Windows 10 Home/ Pro, Windows 11 Home/Pro
- CPU: 1 core
- RAM: 4GB
- DISK: 20GB free disk space

## Step 1

Access the [hivello.com](https://hivello.com/) website, click on the **Download** button at the top right. the system will automatically download to your local machine.

## Step 2

Wait for the downloading process complete.

## Step 3

Once the download is finished, navigate to the download folder. You will find the file named **Hivello....** . Double click on this file or select the option **Run as an administrator**

## Step 4

Follow the introductions provided by the Desktop app. Please note:
- _If WSL (Windows Subsystem for Linux) is not enabled on your local machine, the application will automatically install WSL._

## Step 5

After the Hivello app has been successfully installed, your machine will need to be restarted. Please restart your machine to ensure the app runs smoothly.

## Step 6

After restarting, the Hivello app will automatically run to create Hivello machine, then open the Desktop app screen
