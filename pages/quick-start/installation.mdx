# Installation

This section will guide you through the steps to install the Hivello application

## Prerequisites

- OS: Windows 10 Home/ Pro, Windows 11 Home/Pro
- CPU: 1 core
- RAM: 4GB
- DISK: 20GB free disk space

## Step 1

Access the [hivello.com](https://hivello.com/) website, click on the **Download now** button at the top right. Select the version that is compatible with your machine’s operating system.

## Step 2

Wait for the downloading process complete.

## Step 3

Once the download is finished, navigate to the download folder. You will find the file named **HivelloInstaller** . Double click on this file or select the option **Run as an administrator**
