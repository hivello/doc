# Installation

This section will guide you through the steps to install the Hivello application.

## Prerequisites

- OS: Windows 10 Home/ Pro, Windows 11 Home/Pro
- CPU: 1 core
- RAM: 4GB
- DISK: 20GB free disk space

## Installation

## Step 1

Visit [hivello.com](https://hivello.com/) website, click on the **Register now** button at the top right and download the appropriate application version with your machineâ€™s operating system.

## Step 2

Wait for the downloading process to complete.

## Step 3

Once the download is finished, navigate to the download folder. You will find the file named **HivelloInstaller** . Double-click on this file to run the application.

![Run hivelloInstaller](/images/installation/run-hivelloInstaller.png)

## Step 4

Follow the introductions provided by Hivello Installer.

### 1. Welcome

![Welcome to deepin](/images/installation/welcome-to-deepin.png)

### 2. Downloading Deepin Components

![Downloading Deepin Components](/images/installation/downloading-deepin-components.png)

### 3. Installing Deepin Components

The application will be installed after completing the download of the Deepin components.

![Installing Deepin Components](/images/installation/installing-deepin-components.png)

### 4. Completed Deepin Setup Wizard

Complete Deepin Setup Wizard and launch the application

![Completed Deepin Setup Wizard](/images/installation/completed-deepin-setup-wizard.png)

## Step 5

Sign up for a new account or log in to your existing account if you have already registered. You can then start using the application.

### 1. Sign up a new account

![Sign up](/images/installation/sign-up.png)

### 2. Verify account by email
After verifying your account by clicking on the link that was sent to your email, you need to click on the **Continue** button for the application to display its pages.

![Verify email](/images/installation/verify-email.png)

### 3. Log in

![Log in](/images/installation/login.png)

### 4. Dashboard displayed

The dashboard will be displayed after you successfully logged in. From this step, you can be able to install the chains and get earnings.

![Dashboard displayed](/images/installation/dashboard-displayed.png)

## Step 6

Install the chain and keep it running to earn the passive income.

By selecting the **Install** button on each chain, you will be able to install the DePIN chain on your machine. Keep the installation process running for a while.

![Install the chain](/images/installation/install-the-chain.png)

After the installation is complete, please keep your machine on and connected to the internet to earn rewards.

![After completely installed](/images/installation/after-completely-installed.png)

## Step 7

Add your bank account details on the application so you will be able to receive your earnings payout from Hivello.

![Add your bank account](/images/installation/add-your-bank-account.png)
